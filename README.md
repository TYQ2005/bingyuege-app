冰阅阁(binguege-app)

一款多功能阅读APP，支持书源导入与交互阅读。

功能特性
-📚  书架管理
-🔍  书源导入与管理
-📖  沉浸式阅读体验
-🌙  夜间模式与字体调节

构建说明
本项目使用科尔多瓦构建，通过GitHub操作自动生成安装包。

手动构建
1.安装Node.js和科尔多瓦
2.克隆项目：`吉特克隆https://github.com/TYQ2005/bingyuege-app.git`
3.进入项目目录：`CD冰月格-app`
4.安装依赖：`npm安装`
5.添加安卓平台：`科尔多瓦平台添加安卓`
6.构建安装包：`Cordova build android--发布`
