  用户安装
    1. 访问: https://github.com/TYQ2005/bingyuege-app/releases
    2. 下载: app-release.apk 文件
    3. 安装: 在 Android 设备上安装 (需要 Android 6.0+)
    4. 使用: 打开应用开始阅读

  开发者本地构建
    # 1. 克隆项目
    git clone https://github.com/TYQ2005/bingyuege-app.git
    cd bingyuege-app

    # 2. 安装依赖
    npm install

    # 3. 构建 APK (自动脚本)
    chmod +x build.sh && ./build.sh

    # 或手动构建
    # cordova build android --release

  发布新版本
    # 执行发布脚本
    chmod +x release.sh && ./release.sh

    # 输入版本号 (如: v1.1.0)
    # 自动创建标签并推送
    # GitHub Actions 自动构建
