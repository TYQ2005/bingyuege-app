# 冰阅 APP 版本历史

## [1.0.0] - 2026-02-13

### ✨ 新增功能
- 四种书源导入格式支持 (JSON、XML、文本、URL)
- 完整的书籍管理和阅读功能
- 夜间/白天主题切换
- 阅读进度自动保存
- 章节导航和目录快速定位
- 书架功能和历史记录

### 🔧 技术栈
- Cordova 12.0
- Android SDK 6.0+
- Native HTML5/CSS3/JavaScript

### 📦 发布信息
- 包体积: ~8MB
- 最小 Android: 6.0 (API 23)
- 支持设备: 所有 Android 设备
